# Predictive Risk Analytics

## Overview
This project applies predictive analytics and logistic regression to assess customer risk and identify factors influencing insurance claim outcomes. The analysis includes data cleaning, exploratory data analysis (EDA), feature engineering, and model evaluation to support data-driven decision making.

## Objectives
- Analyse customer and policyholder data
- Identify key risk factors associated with insurance claims
- Build and evaluate a logistic regression model
- Generate actionable business insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Key Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection and engineering
- Logistic Regression modelling
- Model performance evaluation
- Risk factor analysis and interpretation

## Results
The model successfully identified important variables influencing claim outcomes and demonstrated the potential of predictive analytics for risk assessment and business decision making.

## Skills Demonstrated
Data Analysis, Predictive Modelling, Risk Analytics, Statistical Analysis, Machine Learning, Data Visualization, Feature Engineering, Business Intelligence
